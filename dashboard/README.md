## 🚀 Deployment & Installation

1. Navigate to the HealthShield Vercel deployment URL.
2. An **Authorize Node Sync** HUD will appear.
3. Select "Install" to place the HUD directly on your device's home screen.

### Local Development
```bash

# Navigate to health dashboard
cd apps/health-shield-ai/dashboard

# Install dependencies

# Start the HUD
npm run dev
```
